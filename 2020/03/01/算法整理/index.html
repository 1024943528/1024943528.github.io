  • 两数相加（简单）
  1. 获取目标值和每个数组下标的差，利用差值寻找到对应的值
  • 两数相加 (中等)
  1. JSON解析ListNode为字符串从字符串中提取出全部数字部分
  2. 反转后即为对应数字的字符串使用BigInt操作数字
  3. 防止溢出将相加后的结果转换为ListNode
  • 无重复字符的最长串（中等）
  1. 先判断要加入的符号是否存在
  2. 如果存在就把这个符号和之前的符号删除
  3. 添加当前的符号
  4. 获取当前字符串的长度

  • 最长回文子串 （中等）（延伸问题的思路）

  • 最长公共前缀（简单）

  1. 当字符串数组长度为 0 时则公共前缀为空，直接返回
  2. 获取第一个字符串(或者获取最短的一个字符串)
  3. 遍历后面的字符串，依次将其与 ans 进行比较，两两找出公共前缀，最终结果即为最长公共前缀
  4. 如果查找过程中出现了 ans 为空的情况，则公共前缀不存在直接返回
  5. 时间复杂度：O(s)O(s)，s 为所有字符串的长度之和
  • 移除元素 (拷贝覆盖，交换移除)
  1. 主要思路是遍历数组nums，每次取出的数字变量为num，同时设置一个下标ans
  2. 在遍历过程中如果出现数字与需要移除的值不相同时，则进行拷贝覆盖nums[ans] = num，ans自增1
  3. 如果相同的时候，则跳过该数字不进行拷贝覆盖，最后ans即为新的数组长度
  1. 主要思路是遍历数组nums，遍历指针为i，总长度为ans
  2. 在遍历过程中如果出现数字与需要移除的值不相同时，则i自增1，继续下一次遍历
  3. 如果相同的时候，则将nums[i]与nums[ans-1]交换，即当前数字和数组最后一个数字进行交换，交换后就少了一个元素，故而ans自减1

  • 最大子序和 (Math.max-返回最大值)

  1. 动态规划的是x首先对数组进行遍历，当前最大连续子序列和为 sum，结果为 ans
  2. 如果 sum > 0，则说明 sum 对结果有增益效果，则 sum 保留并加上当前遍历数字
  3. 如果 sum <= 0，则说明 sum 对结果无增益效果，需要舍弃，则 sum 直接更新为当前遍历数字
  4. 每次比较 sum 和 ans的大小，将最大值置为ans，遍历结束返回结果
